# TaskApp

O app é uma lista de tarefas você pode adicionar novas tarefas e remover as tarefas já concluidas.

# Tecnologias usadas 🛠

º React Native

º Expo

º JavaScript

º Firebase

º NodeJs

# Pré-requisitos 📱

º NodeJs 12.9

º IDE (VSC ou semelhantes)

º AndroidStudio (Ou celular para emular o app)

º Expo 3.15.2
